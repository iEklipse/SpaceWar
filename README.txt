Game Title: Space War
Author: Tri-Luong Steven Dien


---=== How To Play ===---
W: Move Up
A: Move Left
S: Move Down
D: Move Right

Shoot: Left Click

The goal of the game is to survive the first 30 seconds of the game and defeat the boss.

Destroy the enemy ships to gain points.
Enemy A (V formation) gives 100 points per ship destroyed.
Enemy B (Sinusoidal trajectory) gives 200 points per ship destroyed.
Boss (The body) gives 500 points per hit.

When an enemy wave is completely defeated, a power-up is dropped on the last destroyed ship.
The power-up will upgrade the player ship up to a maximum of 2 upgrades (Max Level is level 3).


---=== How To Read The Code ===---
Comments on top of the script code will tell the reader what does the code does in the game.


---=== How To Compile, Run And Play the Game ===---
Open the game project on Unity
File > Build Settings > Platform: PC, Mac, & Linux Standalone > Build
Chose a destination folder for the game
Open the .exe in the created folder
Play the game !
